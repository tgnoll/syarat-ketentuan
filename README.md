# Syarat & Ketentuan — PeluangCPNS

Halaman Syarat & Ketentuan resmi untuk platform **PeluangCPNS**, di-deploy via Vercel sebagai static page.

## Isi

- Tentang Layanan
- Penggunaan Platform
- Akun Berbayar (termasuk kebijakan no refund)
- Privasi & Data Pengguna
- Perubahan & Penghentian Layanan
- Kontak

## Deploy

Repo ini terhubung ke Vercel. Setiap push ke branch `main` akan otomatis ter-deploy.

**Live URL:** https://syarat-ketentuan.vercel.app
